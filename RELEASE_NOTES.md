# MirrorNovel 更新公告

> 面向用户的功能更新与改进说明

## v1.6.0 (2026-05-30) — 章节规划 & 全文调优

### 📋 章节计划表

生成整本小说时，AI 现在会先制定一份**详细的章节计划表**，细化到每章的核心事件、伏笔设置/回收节点、字数分配。

- **全局规划**：大纲生成后，AI 自动生成章节计划表，确保故事结构完整
- **每章有据可依**：生成每一章时都注入【本章计划】和【章节计划表】，AI 清楚当前进度应该写什么
- **故事状态追踪**：每章注入【当前故事状态】，显示已完成进度、待回收伏笔、剩余章节数
- **防重复生成**：自动提取上一章内容摘要传给下一章，明确指令"禁止与上一章核心事件重复"
- **结局节奏控制**：剩余不足 10 章时提示收紧节奏，最后 3 章要求有力量不烂尾的结局

### 📝 全文调优（修复流水账/重复/伏笔）

完成的小说现在可以一键进行**全文调优**了！入口在小说详情页。

- **AI 分析全文**：自动检测情节重复、流水账章节、未回收伏笔、节奏问题
- **逐章优化重写**：对有问题的章节自动重写修复，没问题的章节做精简润色
- **集成去AI味**：调优完成后自动执行去AI味 + 标点修正
- **一键完成**：点击按钮即可，无需逐章操作

### 🔗 参考小说结构全面生效

- **大纲也参考**：勾选"使用参考结构"后，大纲生成也严格遵循参考小说的剧情走向和章节规划
- **世界观强制覆盖**：自动从参考结构中提取世界观设定，替换用户输入
- **续写继承**：续写时自动复用已保存的参考结构上下文

### ✨ 整本去AI味

小说详情页新增「整本去AI味」按钮，一键对全书所有章节执行 AI 去AI味处理，逐章显示进度。

### 🖥️ Ollama 本机模型选择

- 模型名改为**下拉框选择**，不再需要手动输入
- 支持「从本机刷新」（浏览器直连本地 Ollama）
- 支持「从服务器刷新」
- 手动输入作为后备方案

### 🐛 问题修复与优化

- 修复了 Vite 代理 EPIPE 导致文件上传 503 的问题，改为 nginx 直连后端
- 修复了保活脚本错误重启客户端导致端口冲突的问题
- 修复了异步异常导致 Express headers sent 崩溃的问题
- 后端增加 requestTimeout/headersTimeout 配置
- nginx 增加 proxy_request_buffering 支持大文件上传
- 新增 HTTPS 端口 3459（自签名证书），解决 Chrome/Edge 禁止 HTTP 网站请求本机 Ollama 的问题

## v1.5.0 (2026-05-29) — 参考小说分析增强

### 📄 上传参考小说全面升级

上传参考小说进行结构分析现在支持**完整小说**上传了！

- **取消 4 万字截取**：原来只分析前 4 万字，现在完整分析整本小说
- **最大支持 500 万字**：单本上限提升至 500 万字
- **智能分批处理**：超长小说自动按章节边界拆分，逐块分析后 AI 合并成完整结构报告，不丢失任何剧情细节

### 🐛 问题修复

- 修复了上传超长文本时 AI API 超上下文窗口导致空响应、前端 JSON 解析崩溃的问题
- 前端错误提示现在能正确显示服务端返回的错误信息，不再显示"Unexpected end of JSON input"

## v1.4.0 (2026-05-26) — 轻小说 & 国际化

### 🌸 新增：轻小说生成

期待已久的轻小说功能来啦！现在你可以生成日式 ACGN 风格的轻小说了。

- **6 种轻小说类型**：异世界转生 🌍、校园恋爱 🏫、奇幻冒险 ⚔️、日常系 ☕、战斗异能 💥、科幻未来 🤖
- **角色属性**：设定主角的萌属性（傲娇、天然呆、元气等），AI 会根据属性调整角色性格和对话风格
- **日式文风**：AI 采用轻小说特有的叙事节奏——对话驱动、角色内心独白、动漫式场景描写
- **与蒸馏库联动**：如果蒸馏库中有轻小说参考数据，系统自动匹配并融入生成

---

### ✨ 润色功能升级

润色文本现在是一个独立的页面了，入口在底部导航栏的「✨ 润色」Tab。

- 功能完全不变，入口更清晰
- 生成页顶部 Tab 改为「✍️ 生成小说」和「🌸 轻小说」

---

### 🏷️ 蒸馏库支持轻小说

- 上传蒸馏时可以选择「普通小说」或「轻小说」
- 轻小说蒸馏使用专门的 ACGN 风格分析，提取角色萌属性、对话风格、动画感描写等特征
- 生成时自动匹配蒸馏库中相同类型的参考数据，AI 会学习参考作品的文风

---

### 🌐 国际化：中英双语

从现在起，MirrorNovel 支持中文和 English 两种界面语言！

- **切换方式**：页面底部的群号旁边有语言切换按钮，或前往「我的」页面切换
- **覆盖范围**：所有页面（生成、润色、书架、蒸馏、个人中心等）的全部文本均已翻译
- **类型与标签**：小说类型（都市→Urban）、子类型（末世/废土/生存→Post-Apocalyptic/Survival）、标签（傲娇→Tsundere）等也全部翻译
